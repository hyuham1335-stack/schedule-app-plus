# 일정, 댓글, 유저 관리 프로그램
+ 개요

일정, 댓글, 유저를 관리하는 세션 인증 방식을 사용한 프로그램 개발
---
+ 주요기능

1. 일정 생성/조회/수정/삭제 기능
2. 댓글 생성/조회/수정/삭제 기능
3. 유저 생성/조회/수정/삭제 기능
4. 세션 인증 방식을 통한 로그인/로그아웃 기능
